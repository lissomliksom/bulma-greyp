Other modifications

In addition to changes to typography and colors, the following variables has been tweaked slightly. These changes are personal preference only, and easy to reset.

$dark
Set to $black-bis instead of $grey-darker.
$radius-large
Radius on boxes. Set to 4px, same radius as buttons.
$box-shadow
Slightly softer shadows.
$box-padding
Slight increase to allow for more space when using tabs inside boxes.
